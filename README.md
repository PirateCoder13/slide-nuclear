# Slide Nuclear

Apresentacao web interativa sobre fusao nuclear em formato de slides, com visual sci-fi, animacoes de sistemas e navegacao por teclado.

## Visao Geral

- Tema: Reactor Boot Sequence / Fusão Nuclear
- Idioma principal do conteudo: portugues (pt-BR)
- Navegacao entre seções com setas do teclado
- Efeitos visuais: parallax, particulas, terminal ao vivo, cards animados e efeito cinematico de explosao

## Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Vite
- Chart.js

## Estrutura

- `index.html`: estrutura da apresentacao
- `style.css`: estilos, temas e animacoes
- `main.js`: logica dos efeitos, interacoes e navegacao

## Scripts NPM

- `npm run dev`: ambiente de desenvolvimento
- `npm run build`: build de producao
